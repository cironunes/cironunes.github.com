We’ll agree that AngularJS is awesome. One of the most handy features of the framework is the data-binding mechanism that avoid us to write a lot of code for wiring up our models and views together.

What you should keep in mind is that the data-binding awesomeness is based on $digest cycles, which are a basic dirty checking (if there’s a change in the value, then fires the change event).
